## Package manager

[Homebrew](https://docs.brew.sh/) is available on both macOS and Linux.

Common commands:

* Install a package, such as `curl`: `brew install curl`
* List packages whose name includes a string, such as `ssh`: `brew search ssh`
* Get information about a package, such as `curl`: `brew info curl`

Repository integration:

* `Brewfile` - formulae (packages) used by this project
