# Software stack specifications

This repository is an [Agent Skill](https://agentskills.io/specification) that
teaches an AI agent my software stack choices.

Example installation in a repository that uses
[gemini-cli](https://github.com/google-gemini/gemini-cli).

```bash
git submodule add https://github.com/pwnall/pwnall-stack-specs.git .gemini/skills/pwnall-stack-specs
```
